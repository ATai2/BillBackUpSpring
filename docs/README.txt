二。功能概述
1.医院信息系统（HIS）数据监控同步系统
在服务器（可能是HIS系统服务器，可能是独立服务器）中部署“HIS系统数据监控同步系统”，
实时同步，启动多个thread对于各表每隔100ms调用存储过程，检测是否有数据更新。
有更新数据，HIS→凭证制作系统，拷贝（SQLBulkCopy）数据库（sqlserver2005）数据。

2.凭证备份系统
定期（每天凌晨1点）执行任务，把凭证制作系统中相关数据备份到我方服务器数据库内，并把制作完成的TIFF文件拷贝到我方服务器中。

3.发票打印系统
打印票据，编号需与HIS系统一致，打印过的发票需作标示，需要身份证读卡器，社保卡读卡器。

4.发票查询系统
与医院联络，在医院现有网站中开辟一块区域链接到我方发票查询网站。
通过票据编号 + 名称，查询发票，成功显示TIFF文件图片。
防恶意攻击，同一IP查询次数限制等。

5.补开发票系统
入库系统增加补开发票功能，第一次打印记录打印次数，入库后记录打印标示，
财务处计算机管理打印可以调取发票，补开发票。
需要，备注补开发票理由，补开发票人员名称等。
